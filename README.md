# Career-Twin
### https://careertwin-beta.vercel.app/dashboard.html

CareerTwin is a career guidance and skill development platform designed to help students and aspiring professionals make informed career decisions.

The platform provides personalized career exploration through skill assessment, roadmap generation, project recommendations, resume analysis, and progress tracking.

Features

- User Registration & Authentication
- Career Roadmap Generation
- Skill Gap Analysis
- Resume Analysis
- Career-Oriented Quizzes
- Personalized Project Recommendations
- Progress Tracking Dashboard
- User Profile Management

Tech Stack

Frontend

- HTML5
- CSS3
- JavaScript

Backend (In Progress)

- Python
- Flask
- SQLAlchemy
- Flask-Login
- MySQL

Project Vision

CareerTwin aims to serve as a digital mentor for students, helping them discover suitable career paths, identify missing skills, and build a structured learning journey through interactive assessments and personalized recommendations.

Future Enhancements

- AI-powered career recommendations
- Advanced resume analysis
- Learning path personalization
- Industry-specific skill tracking
- Analytics dashboard
- Internship and job recommendation system
